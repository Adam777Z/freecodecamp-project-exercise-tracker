Exercise Tracker

Create a New User

POST /api/exercise/new-user

Add exercises

POST /api/exercise/add

GET user's exercise log: GET /api/exercise/log?{userId}[&from][&to][&limit]
{ } = required, [ ] = optional
from, to = dates (yyyy-mm-dd); limit = number

GET all users: /api/exercise/users
